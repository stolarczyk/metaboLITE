What is metaboLITE?

We aimed to develop a tool to decouple learning to program and learning about genome-scale metabolic modeling. metaboLITE is a web-based application that visualizes and implements two simple reconstructions and an analytic framework used to analyze metabolic reconstructions.

Biologist-friendly

metaboLITE is presented from a biologistâ€™s point of view. With metaboLITE, the user can perform three in silico simulations, presented as experiments: (1) changing growth media, (2) knocking out genes, and (3) under/overexpressing enzymes

Intuitive interface

From now on, you can use graphical user interface to interact with the models via implemented functionalities. No coding skills required!

Easy installation

metaboLITE can be installed through an online download using Docker graphical user interface - Kitematic. Available for all major platforms: Windows, macOS and Linux!
